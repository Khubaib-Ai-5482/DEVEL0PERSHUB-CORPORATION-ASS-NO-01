# Machine Learning Mini Projects

This repository showcases several beginner-friendly machine learning projects. These tasks focus on **data exploration, visualization, cleaning, classification, regression, and model evaluation** using Python.

Technologies used:

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Task 1: Exploring and Visualizing the Iris Dataset

**Objective:**  
Understand how to load, inspect, summarize, and visualize a dataset.

**Dataset:**  
Iris Dataset

**Description:**  
- The dataset was loaded and explored to check its structure, columns, and first few rows.  
- Visualizations such as scatter plots, histograms, and box plots were used to identify patterns and relationships between features.  
- The analysis helped in understanding how different species of iris flowers vary in sepal and petal measurements.

**Skills Developed:**  
- Data loading and inspection using pandas  
- Basic statistical summarization  
- Data visualization using matplotlib and seaborn  

---

## Task 2: Credit Risk Prediction

**Objective:**  
Predict whether a loan applicant is likely to default.

**Dataset:**  
Loan Prediction Dataset

**Description:**  
- Missing values were identified and handled to ensure clean data.  
- Exploratory visualizations were performed to understand key features such as loan amount, income, and education level.  
- A classification model was trained to predict loan default.  
- Model performance was evaluated using accuracy and confusion matrix, allowing identification of predictive strengths and weaknesses.

**Skills Developed:**  
- Data cleaning and handling missing values  
- Exploratory Data Analysis (EDA)  
- Binary classification modeling  
- Model evaluation techniques  

---

## Task 3: Customer Churn Prediction

**Objective:**  
Identify customers who are likely to leave the bank.

**Dataset:**  
Churn Modeling Dataset

**Description:**  
- The dataset was prepared by encoding categorical variables such as gender and geography.  
- A classification model was trained to predict churn.  
- Feature importance analysis highlighted which variables most strongly influence customer retention or departure.

**Skills Developed:**  
- Categorical data encoding  
- Supervised machine learning  
- Feature importance analysis  

---

## Task 4: Predicting Insurance Claim Amounts

**Objective:**  
Estimate medical insurance claim charges using regression.

**Dataset:**  
Medical Cost Personal Dataset

**Description:**  
- Linear regression models were trained to predict insurance charges based on features such as age, BMI, and smoking status.  
- Visualizations showed correlations between variables and the target value.  
- Model performance was evaluated using metrics like mean absolute error (MAE) and root mean squared error (RMSE).

**Skills Developed:**  
- Regression modeling  
- Correlation analysis and visualization  
- Model evaluation using MAE and RMSE  

---

## Task 5: Personal Loan Acceptance Prediction

**Objective:**  
Predict which customers are likely to accept a personal loan offer.

**Dataset:**  
Bank Marketing Dataset

**Description:**  
- Basic data exploration identified trends in features such as age, job type, and marital status.  
- A classification model was trained to predict loan acceptance.  
- Analysis of results provided insights into which customer segments are most likely to respond positively to loan offers.

**Skills Developed:**  
- Data exploration and visualization  
- Classification modeling  
- Extracting actionable business insights from data

These projects demonstrate the full workflow of a data science project:
